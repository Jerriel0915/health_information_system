# 数据库初始化说明

## 前置条件

1. 安装 MySQL 8.0+
2. 确保 MySQL 服务正在运行
3. 记下你的 MySQL 用户名和密码（默认为 root / 空密码）

## 步骤

### 第一步：创建数据库

打开 MySQL 命令行或 Navicat 等工具，执行：

```sql
CREATE DATABASE IF NOT EXISTS health_data_manager DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 第二步：导入 RuoYi 系统表

项目根目录下有一个 RuoYi 框架的 SQL 文件，需要先导入。路径为 `server/sql/ry_20260417.sql`（如文件名不同请查看 `server/sql/` 目录下最新的 `.sql` 文件）。

命令行导入：
```powershell
mysql -u root -p health_data_manager < server/sql/ry_20260417.sql
```

或使用 Navicat 等工具直接运行该 SQL 文件。

### 第三步：导入业务数据

导入我们自己的业务表和数据：

```powershell
mysql -u root -p health_data_manager < server/init_db.sql
```

或在 Navicat 中运行 `server/init_db.sql` 文件。

### 第四步：修改数据库密码

打开 `server/ruoyi-admin/src/main/resources/application-druid.yml`，
找到 `password: l123456789` 改为你自己的 MySQL 密码。

## 验证

导入完成后，数据库中应包含以下表：

**RuoYi 系统表（部分）：**
`sys_user`、`sys_menu`、`sys_role`、`sys_config` 等

**业务表（共7张）：**
`dim_region`、`dim_population`、`medical_institution`、`medical_staff`
`medical_bed`、`medical_service`、`medical_cost`

确认有数据：
```sql
USE health_data_manager;
SELECT COUNT(*) FROM medical_institution;  -- 应有 78 条
SELECT COUNT(*) FROM medical_staff;        -- 应有 5206 条
SELECT COUNT(*) FROM medical_service;      -- 应有 50000 条
```
